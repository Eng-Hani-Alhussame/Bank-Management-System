# 💹Capital One Bank is a banking management system developed using C# and Windows Forms App (WFA) on the .NET Framework. It uses SQL Server as the database to store and manage customer data, accounts, and financial transactions.

## 🚀 Key Features

· Customer management (Add – Edit – Delete – Search)
· Deposit, Withdraw, and Transfer between accounts
· View transaction history (account statement)
· User authentication (Login / Logout)

---

## 🛠️ Technologies Used

· Language: C#
· Framework: .NET Framework (Windows Forms App)
· Database: Microsoft SQL Server
· Data Access: ADO.NET (SqlConnection, SqlCommand, SqlDataAdapter)
